IPL Analytics

IPL Analytics Web App

Team performance, broken down by every player who actually took the field.

This app analyzes the last three IPL seasons in your local dataset and now layers in the official live IPL feeds for the current season, so you can compare historical performance, current player form, the live table, and title odds in one place.

Dataset window 2023 to 2025
Online enrichment Official IPL live feeds + ESPN Cricinfo profile links
Live layer Current table, squad form, fixtures, and winner odds

Sources

Data inputs

  • Local match summaries: csv/Match_Info.csv
  • Local ball-by-ball feed: csv/Ball_By_Ball_Match_Data.csv
  • Online player directory: csv/2024_players_details.csv image/profile links
  • Official IPL 2026 points table
  • Official IPL 2026 stats page
  • Official IPL competition feed
  • Official IPL teams directory

Open this app through a local web server so the browser can fetch the CSV files correctly.
